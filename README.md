# Ponderada OpenChoreo

Entrega individual da atividade de validação local do OpenChoreo para a disciplina de arquitetura.

## Objetivo

Validar a instalação local do OpenChoreo com Docker, acessar a interface web, registrar evidências reais e, se possível, publicar a aplicação de exemplo da documentação oficial.

## Estrutura

- `docs/instalacao-openchoreo.md`: documentação principal da execução.
- `docs/evidencias/`: evidências de terminal e interface.

## Evidências iniciais

As primeiras evidências coletadas incluem:

- sistema operacional do host;
- versão do Docker;
- memória e CPUs do host;
- memória e CPUs disponíveis no ambiente Docker.
